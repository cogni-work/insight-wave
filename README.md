# cogni-works

Claude Code marketplace for cogni plugins.

## Plugins

| Plugin | Description |
|--------|-------------|
| [cogni-claims](./cogni-claims) | Claim verification and management system |

## Install

```bash
git clone --recurse-submodules https://github.com/cogni-work/cogni-works.git
```

To update all submodules:

```bash
git submodule update --remote --merge
```
