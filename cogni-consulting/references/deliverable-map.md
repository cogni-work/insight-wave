# Deliverable Map

Each vision class produces a different set of deliverables. This map defines the default package for each class. Consultants can add or remove deliverables during setup.

## Deliverable Catalog

| Deliverable | Description | Formats | Source Phase |
|---|---|---|---|
| Strategic Options Brief | Ranked alternatives with evaluation criteria | PPTX, DOCX | Deliver |
| Business Case | Financials, assumptions, sensitivity analysis | XLSX + DOCX | Deliver |
| Decision Board | Visual option map with recommendation | Excalidraw, SVG | Deliver |
| Executive Summary | One-pager for leadership alignment | PPTX, PDF | Deliver |
| Action Roadmap | Phased implementation plan with milestones | PPTX, XLSX | Deliver |
| TIPS Landscape | Trend/implication/possibility map | MD, Excalidraw | Discover |
| Portfolio Snapshot | Competitive positioning and value wedge | XLSX, PPTX | Develop |
| Claim Verification Log | Audit trail of verified/flagged assertions | MD, XLSX | Deliver |
| Market Assessment | Market size, dynamics, and entry barriers | DOCX, PPTX | Discover |
| Scenario Matrix | 2×2 scenario analysis with implications | PPTX, Excalidraw | Develop |
| Cost Reduction Playbook | Prioritized savings opportunities with implementation | XLSX, DOCX | Deliver |
| Transformation Roadmap | Current→target state with transition phases | PPTX, Excalidraw | Deliver |
