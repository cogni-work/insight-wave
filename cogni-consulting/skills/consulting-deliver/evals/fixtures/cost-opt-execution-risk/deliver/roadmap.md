# Action Roadmap

## Phase Overview

| Phase | Timeline | Focus | Budget |
|---|---|---|---|
| 1: Quick Wins | Months 1-3 | AWS rightsizing, FinOps setup, DC2 migration planning | €400K |
| 2: Infrastructure | Months 3-6 | DC2→cloud DR migration, DC3 consolidation, BaFin compliance | €280K |
| 3: Support Transformation | Months 4-8 | Self-service portal, Tier 1 automation, staff retraining | €288K |

## Phase 1: Quick Wins (Months 1-3)

**Goal**: Realize €775K annual savings from AWS optimization; establish FinOps governance

### Workstreams
- AWS rightsizing: terminate/resize dev/test instances (120+ instances audit)
- Reserved instance optimization: convert on-demand to reserved where stable
- FinOps tooling: implement cloud cost management platform
- DC2 migration planning: design cloud DR architecture
- BaFin pre-consultation: initiate dialogue on cloud DR compliance

### Resources
- 3 IT staff (cloud team, full-time on optimization)
- 1 FinOps consultant (external, 3 months)

### Decision Gate → Phase 2
- AWS cost reduction ≥€50K/month demonstrated
- Cloud DR architecture approved by IT security
- FinOps governance framework operational

## Phase 2: Infrastructure Consolidation (Months 3-6)

**Goal**: Migrate DC2 to cloud DR; consolidate DC3 branch processing

### Workstreams
- DC2 server migration to AWS (60 servers, 3 months)
- DC3 branch processing migration to centralized infrastructure (40 servers)
- Network reconfiguration for 12 branches
- BaFin notification and compliance documentation
- Decommission DC2 and DC3 physical infrastructure

### Resources
- 5 IT staff (infrastructure team + 2 from cloud team)
- Migration partner (external, 3 months)
- BaFin compliance consultant (10 days)

### Decision Gate → Phase 3
- DC2 decommissioned, cloud DR operational
- DC3 consolidated, branch connectivity verified
- BaFin compliance confirmed
- No unplanned outages during migration

## Phase 3: Support Transformation (Months 4-8)

**Goal**: Deploy self-service portal; automate Tier 1; retrain staff

### Workstreams
- Self-service portal deployment (ServiceNow Employee Center)
- Password reset automation (SSPR)
- Standard software request automation
- Hardware request workflow automation
- Staff retraining: 8 Tier 1 support staff → Tier 2 capabilities
- Communication campaign: drive portal adoption across 4,200 endpoints

### Resources
- 2 IT staff (service management)
- ServiceNow implementation partner (4 months)
- Training provider (retraining program, 3 months)

### Decision Gate → Program Close
- Self-service portal adoption ≥40% of total tickets
- Tier 1 ticket volume reduced ≥40%
- All retrained staff certified and operational in Tier 2 roles
- Total annual savings on track (±15% of €2,000K target)
