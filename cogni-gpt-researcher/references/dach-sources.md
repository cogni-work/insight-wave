# DACH-Specific Sources for Web Research

When project language is `de`, section-researcher and deep-researcher should include DACH-specific searches alongside standard English queries. This reference provides authoritative German-language and DACH-region sources that English-only searches miss.

## Why Bilingual Search Matters

German-language sources capture DACH market insights, regulatory context, and Mittelstand perspectives that rarely appear in English results. The strategy is always bilingual: English queries for global coverage, German queries for regional depth.

## 8. Bilingual Query Construction

When language=de, generate queries using this pattern:

| Query Type | Language | Example |
|------------|----------|---------|
| Global reach | English | `"quantum computing" impact cryptography 2026` |
| DACH perspective | German | `"Quantencomputing" Auswirkungen Kryptographie Deutschland 2026` |
| Site-specific | German | `site:fraunhofer.de Quantencomputing Sicherheit Studie 2026` |
| DACH regional | English | `"quantum computing" Germany Austria Switzerland regulation 2026` |

**German query tips:**
- Use compound nouns: "Digitalisierungsstrategie", "Energiewende", "Fachkräftemangel"
- Include "Deutschland", "Österreich", "Schweiz" or "DACH" for regional focus
- Use industry-specific German terms alongside English equivalents
