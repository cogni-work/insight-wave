# ICT Taxonomy

57 standard service categories across 8 dimensions (0-7). This is the single source of truth for the B2B ICT Portfolio taxonomy used by the `ict-scan` skill, `portfolio-dashboard`, and the `bridge` skill.

For cross-category overlap rules, see [cross-category-rules.md](cross-category-rules.md).

## 0. Provider Profile Metrics (6 categories)

| ID | Category | Description |
|----|----------|-------------|
| 0.1 | Financial Scale | Annual revenue/turnover, market cap, revenue growth trends |
| 0.2 | Workforce Capacity | Total employees, IT specialists, regional workforce distribution |
| 0.3 | Geographic Presence | HQ location, delivery centers, service countries, data center locations |
| 0.4 | Market Position | Market share, industry rankings, analyst ratings, key reference clients |
| 0.5 | Certifications & Accreditations | ISO certifications, industry accreditations, regulatory compliance |
| 0.6 | Partnership Ecosystem | Hyperscaler tiers (AWS Premier, Azure Expert MSP), strategic alliances |
