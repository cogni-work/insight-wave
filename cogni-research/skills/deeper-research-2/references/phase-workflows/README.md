# Phase Workflows - Deeper Research 2 (Enrichment)

This directory contains the phase workflow reference files for deeper-research-2 (enrichment phases 4-7).

## Phase Files

| Phase | File | Purpose |
|-------|------|---------|
| 4 | `phase-4-source-creation.md` | Source creation and validation |
| 5 | `phase-5-knowledge-extraction.md` | Concept extraction and megatrend clustering |
| 6 | `phase-6-publisher-citation.md` | Publisher and citation generation |
| 7 | `phase-7-claims-creation.md` | Claims verification with confidence scores |

## Usage

Each phase file must be read BEFORE executing that phase. The files contain:

- Required TodoWrite expansion templates
- Phase entry verification gates
- Step-by-step implementation details
- Self-verification checkpoints

## Phase Flow

```
Phase 4 (Sources) -> Phase 5 (Knowledge) -> Phase 6 (Citations) -> Phase 7 (Claims)
```

Each phase has strict entry gates that verify the previous phase completed successfully.
