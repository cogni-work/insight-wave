# Industry Taxonomy for Trend Scout

This taxonomy provides a standardized industry classification with subsectors for TIPS scouting. All entries are bilingual (English/German).

## Usage

When presenting to user, show:

1. Top-level industry with number
2. Subsectors as lettered options

**Example display:**

```text
1. Manufacturing / Fertigung
   a) Automotive / Automobil
   b) Machinery / Maschinenbau
   c) Electronics / Elektronik
```

**User can respond with:**

- Number + letter: `1a`, `1.a`, `1-a`
- Full name: `Manufacturing - Automotive`
- German name: `Automobil`, `Fertigung - Automobil`
- Natural language: `automotive industry`, `Automobilindustrie`

## Parsing Logic

To parse user selection:

```bash
cat > /tmp/ts-parse-selection.sh << 'SCRIPT_EOF'
#!/usr/bin/env bash
set -eo pipefail

# Input: user_selection (e.g., "1a", "1.a", "manufacturing - automotive", "Automobil")
# Output: industry_slug, subsector_slug, industry_en, industry_de, subsector_en, subsector_de

# Pattern 1: Number + Letter
if [[ "$user_selection" =~ ^([0-9]+)[.-]?([a-z])$ ]]; then
  industry_num="${BASH_REMATCH[1]}"
  subsector_letter="${BASH_REMATCH[2]}"
  # Map number to industry, letter to subsector
fi

# Pattern 2: Full name match
# Compare against taxonomy entries (case-insensitive)

# Pattern 3: Natural language
# Use LLM inference to map to closest match
SCRIPT_EOF
chmod +x /tmp/ts-parse-selection.sh && bash /tmp/ts-parse-selection.sh
```
