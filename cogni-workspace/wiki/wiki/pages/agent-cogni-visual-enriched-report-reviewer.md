> One of the agents inside [[plugin-cogni-visual]]. Model tier: **sonnet** — see [[concept-agent-model-strategy]].

Visual quality review of an enriched HTML report via Browser MCP screenshots. Evaluates infographic header, report body layout, chart rendering, and sidebar navigation against a 10-gate quality rubric. Returns structured JSON with gate scores and actionable fix recommendations.
