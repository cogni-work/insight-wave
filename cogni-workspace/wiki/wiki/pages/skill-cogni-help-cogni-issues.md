> One of the skills inside [[plugin-cogni-help]].

File and track GitHub issues (bugs, feature requests, change requests, questions) against insight-wave ecosystem plugins using browser automation (claude-in-chrome). Guides users through a short consultation to capture the right details, resolves the target plugin's repository automatically, drafts issues from templates, creates them via browser automation on github.com, and tracks them locally.
